STYLING

There are maily three types of styling methods that we can apply using CSS :

  • inline Styles Inline styling can be one done the body section of the html file. It can be used to style just one element at a time
  • Internal Styles It is mainly written at the head section of the html file It can be applied at the whole document not just a single element.
  • External Styles In external styling a diffrenet external file is used to style the webpage. This CSS file comes with an extension of .css

    SELECTORS

    A CSS selector is used to select a specific portion of the html webpage that the user wants to style

    Class selectors

    The .class selector is used to style the element having a specific class attribute.

    ID selectors

    The #id selector is used to style the element using a specified id.

    Pseudo-class selectors

    If an element has a special state then Pseudo-class selectors are used.

    EXAMPLES

    It can be used to:

  • Style an element when a user mouses over it
  • Style visited and unvisited links differently
  • Style an element when it gets focus

    • Some examples of selectors are given below:

    1. Element selector p { background-color: gold; }
    2. #id selector #special { background-color: gold; }
    3. .class selector .createBoldAndDoubleText { font-weight: bold; font-size: 2em; }
    4. Attribute selectors
      • Example-1: input[type="checkbox"] { color: gold;}
      • Example-2: a[href="https://www.google.com"] { color: gold; }
    5. Descendant selector
    6. Sibling selectors div ~ p { background-color: gold; }
      • Example-1: .class li{background-color: gold;}
      • Example-2: ul li a { background-color:gold; }
    7. Adjacent selectors div + p { background-color: gold; }
      • Example-1: li:nth-of-type(7) { color: gold;}
      • Example-2: li:nth-of-type(odd) { color: gold;}
    9. nth-child
      • Example-1: ul:nth-child(2) { color: gold; }
      • Example-2: ul:nth-child(even) { color: gold; }
    10. The star selector * * { margin: 0; padding: 0; }
